Octal to Decimal and vice versa





To convert an octal number to a decimal number, multiply each digit with

the reducing power of eight.





EXAMPLE 1

Octal to Decimal

Convert 2158 to decimal

2 x 82 = 2 x 64 = 128

1 x 81 = 1 x 8 = 8

5 x 80 = 5 x 1 = 5

Then add all the results: 128 + 8 + 5 = 141

Therefore we can conclude that 2158 = 14110
