Binary-Decimal and vice versa





To convert a binary number to a decimal number, perform a multiplication operation

on each digit of the binary number from right to left with powers of 2

beginning with 0 and add the results to get the decimal number





EXAMPLE 1

Binary to Decimal

Let's consider a binary number 1101. We need to convert this binary number to a decimal number

1 x 23=8

1 x 22=4

0 x 21=0

1 x 20=1

Then add the result, 8+4+0+1 = 13

Therefore we conclude that 11012 = 1310




Various method are used to convert decimal to binary

One method for converting decimal to binary is to divide the given decimal number

by 2 recursively. The remainders are then recorded until the final quotient is zero.

Following this, the remainders are written backwards to obtain

the binary value of the given decimal number.
